Accessibility

DevPT aims to make its public website usable for visitors using keyboard navigation, screen readers, zoom, reduced motion settings, and mobile devices.

Current approach

The public site uses semantic headings, descriptive links, visible focus states, responsive layouts, reduced-motion handling, and text alternatives for meaningful images.

Known limitations

Some linked simulation tools include highly visual or interactive clinical interfaces. Those tools are still being developed and may have separate accessibility work in progress.

Feedback

If something on this site is difficult to use, email danstone.pt@gmail.com with the page, browser, assistive technology if relevant, and the problem you encountered.
